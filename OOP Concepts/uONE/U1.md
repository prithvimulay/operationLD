# Objects 
- these are real world entity like a car, a person, a bank account, etc.
- anything that can be described as a thing with characteristics (state) and capabilities (behavior) is an object.
Characteristics are also known as attributes/properties/state.
eg: Car is a object having characteristics like color, brand, model, etc.
Behaviors are known as methods/actions.
eg: Car is a object having behaviors like start(), stop(), brake(), drive() etc.

Objects have three main characteristics:

1. State/Attributes/Properties: The data or attributes of the object. In the Person class, the name and age are the object's state.
2. Behavior/Methods/Functions: The methods and functions that the object can perform. In the Person class, the display method is mentioned here as an example of behavior, and it will be implemented later in this lesson.
3. Identity: A unique identifier distinguishes the object from others, even if they have the same state. This is handled by the memory address in JavaScript.

An object is thus a concrete instance of a class that includes state, behavior, and identity.
